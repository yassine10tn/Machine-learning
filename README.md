# Machine Learning Algorithms

A collection of Machine Learning algorithms implemented in Python using real-world datasets.

## Algorithms Implemented

### Unsupervised Learning
- **K-Means Clustering** — Mall customer segmentation using Annual Income, Spending Score, and Age
